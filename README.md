# Shader Generator

**Create Custom Shaders Easily in Unity**

Generate professional shaders without writing code from scratch.

## Features

- **8 Shader Types** - Toon, Hologram, Dissolve, Outline, Rim, Water, Gradient, Emission
- **URP Compatible** - Works with Universal Render Pipeline
- **Customizable** - Adjust colors and parameters
- **Material Creation** - Create materials from generated shaders
- **Clean Code** - Generates well-structured HLSL code

## Installation

1. Import this package into your Unity project
2. Go to **Tools > Shader Generator**

## Usage

1. Select shader type
2. Set colors and parameters
3. Click "GENERATE SHADER"
4. Optionally create a material

## Requirements

- Unity 6.3 LTS or later
- Universal Render Pipeline (URP)
